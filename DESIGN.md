# DESIGN GUIDE

## Design Philosophy
- **미니멀 + 학술적**
- 불필요한 장식 없이 콘텐츠 중심
- 논문 리더가 집중할 수 있는 조용한 UI
- 참고 레퍼런스: Perplexity AI, Elicit, Notion

---

## Layout

### 메인 화면 (초기 진입)
- 화면 정중앙에 입력창 하나만 표시 (ChatGPT 스타일)
- 로고/서비스명 + 한 줄 설명 + URL 입력창 + 옵션(선택) + 분석 버튼
- 배경은 순백색, 입력창에만 하늘색 포인트

```
┌─────────────────────────────────────┐
│                                     │
│        Paper Reader                 │  ← 서비스명 (상단 중앙)
│  논문 링크를 붙여넣으세요             │  ← 서브타이틀
│                                     │
│  ┌─────────────────────────────┐   │
│  │  https://arxiv.org/...  [→] │   │  ← URL 입력창 + 버튼
│  └─────────────────────────────┘   │
│                                     │
│  [ 요약 깊이 ▾ ]  [ 수준 ▾ ]        │  ← 선택적 옵션 드롭다운
│                                     │
└─────────────────────────────────────┘
```

### 결과 화면
- 왼쪽: 사이드바 (목차 - 섹션 네비게이션)
- 오른쪽: 결과 콘텐츠 (카드 단위로 섹션 구분)
- 스트리밍 방식으로 결과가 순서대로 등장

---

## Component Style

### 입력창
- 둥근 모서리: `border-radius: 12px`
- 테두리: `1px solid #E2E8F0`
- 포커스 시: 하늘색 테두리 `#60B4E8`
- 내부 패딩: `16px 20px`
- 전송 버튼: 하늘색 배경 `#60B4E8`, 흰색 아이콘

### 결과 카드
- 배경: `#F8F9FA`
- 테두리: `1px solid #E2E8F0`
- 섹션 제목: 하늘색 좌측 바 강조
- 패딩: `24px`

### 버튼
- Primary: 하늘색 `#60B4E8` + 흰색 텍스트
- Secondary: 흰색 배경 + 회색 테두리

---

## Animation
- 입력창 포커스: 테두리 색 전환 (0.2s ease)
- 결과 등장: 스트리밍 텍스트 또는 fade-in (섹션 단위)
- 로딩: 하늘색 점 3개 애니메이션 (···)

---

## Responsive
- 모바일: 단일 컬럼, 사이드바 숨김
- 태블릿 이상: 사이드바 + 메인 2열 레이아웃
